<div align="center"><strong>AdminFlow</strong></div>
<div align="center">Modern Admin Dashboard built with Next.js App Router</div>

## Overview

AdminFlow is a clean and modern admin dashboard using the following stack:

- Framework - [Next.js (App Router)](https://nextjs.org)
- Language - [TypeScript](https://www.typescriptlang.org)
- Auth - Custom credentials-based authentication (cookie session)
- Styling - [Tailwind CSS](https://tailwindcss.com)
- Components - [Shadcn UI](https://ui.shadcn.com/)
- Formatting - [Prettier](https://prettier.io)

## Features

- Username/password login with Remember Me support
- Collapsible sidebar navigation
- Products management with search and pagination
- Orders listing with status badges
- Customers page
- In-memory mock data (no database required)

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
